<h1>Mr. CapMan FakeStore</h1>

Projeto simples em React, criando uma FakeStore para o Desafio Final do Projeto Desenvolve.

<h2>Como rodar</h2>

Abra o terminal na pasta onde o projeto foi salvo e digite os comandos abaixo
```bash
npm install
npm run dev
```

Depois, abra o endereço exibido no terminal.

*É preciso ter conexão de internet pra funcionar, pois é necessário conectar a API
